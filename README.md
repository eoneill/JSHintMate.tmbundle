# JSHintMate

Easy JSHinting for TextMate2

## Installation

```sh
mkdir -p ~/Library/Application\ Support/Avian/Bundles
cd ~/Library/Application\ Support/Avian/Bundles
git clone git://github.com/eoneill/JSHintMate.tmbundle.git
```
